## 🚀 How to Test

1. **Open index.html** in your browser
2. **Type some text** in the editor
3. **Watch statistics** update in real-time
4. **Try Quick Actions** on the right sidebar
5. **Click Transform tab** and try any tool
6. **Click Extract tab** and test extraction
7. **Click Developer tab** and try dev tools
8. **Click Compare tab** and compare two texts
9. **Press Ctrl+K** for command palette
10. **Press Ctrl+/** for focus mode
11. **Click theme toggle** for dark mode
12. **Try import/export** with different formats
