# 🔎 MianScan — Website SEO & Intelligence Analyzer

> Scan any URL and get complete website intelligence in seconds.

MianScan is a fast, powerful website analysis tool that extracts SEO data, tags, ranking insights, contact info, and technical details from any website. Built for SEO experts, freelancers, developers, and content creators.

🌐 **Live Demo:** [mianhassam96.github.io/MianScan](https://mianhassam96.github.io/MianScan/)

---

## 🚀 Features

### 1. Website Overview
Quick snapshot of the site — title, meta description, domain, language, page type, and word count.

### 2. SEO Score
Automatic SEO score based on title, meta description, headings, alt tags, keywords, links, and canonical. Shows overall SEO health.

### 3. Domain Authority / Page Authority
Displays Domain Authority (DA), Page Authority (PA), and domain strength. Useful for competitor analysis.

### 4. Website Ranking Data
Global rank, country rank, category rank, and estimated traffic.

### 5. Keywords & Tags Extractor
Extracts meta keywords, heading keywords, content keywords, and SEO tags. Perfect for content research.

### 6. Headings Structure
SEO heading breakdown — H1, H2, H3 counts and heading hierarchy.

### 7. Links Analyzer
Internal links, external links, follow links, and nofollow links. Useful for SEO audits.

### 8. Images SEO Analyzer
Total images, missing alt tags, optimized images, and lazy loading status.

### 9. Contact Information Finder
Extracts email addresses, phone numbers, WhatsApp, and social media links. Great for freelancer outreach.

### 10. CTA / Buttons Analyzer
Detects primary CTAs, button text, and anchor links. Helpful for UX & conversion research.

### 11. Tech Stack Detector
Detects CMS, frameworks, libraries, and analytics tools — WordPress, React, Next.js, Tailwind, Google Analytics, and 28+ more.

### 12. Performance Stats
Page size, scripts count, CSS files, and images count.

### 13. Meta Tags Analyzer
Extracts meta title, meta description, Open Graph tags, Twitter cards, and canonical URL.

### 14. Indexing & Robots Checker
Detects robots.txt, noindex/index, canonical, and robots meta tags. Important for SEO indexing.

### 15. Compare Two Websites
Compare two domains side by side — SEO score, DA/PA, ranking, keywords, links, images, headings, meta tags, and tech stack. Perfect for competitor analysis.

## ⚡ How It Works

1. Enter a website URL
2. Click **Scan**
3. MianScan fetches and analyzes the page
4. View the full report instantly
5. Compare with a competitor

---

## 🛠 Tech Used

- Vanilla JavaScript
- DOMParser
- CORS Proxy
- Regex extraction
- Client-side only — no backend, no signup

Provide a simple, fast, and powerful website analysis tool without heavy APIs or complex setup.
