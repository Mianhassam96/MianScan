# MianScribe Premium - Quick Guide

## 🚀 Getting Started

1. **Open** `index.html` in your browser
2. **Start typing** or paste text
3. **Explore** 50+ tools from sidebar
4. **Export** in 7 formats

## 📤 Upload & Download

### Upload Files
- Click **📤 Upload** button (top toolbar)
- Supported: TXT, HTML, RTF, PDF, DOC, DOCX, JSON, Markdown
- Content loads instantly
- PDF text extraction supported
- WordPad (.rtf) files fully supported

### Download Files
- Click **Download** dropdown
- Choose format: TXT, HTML, PDF, DOC, RTF, JSON, MD
- File downloads automatically

### RTF/WordPad Support ✅
- **Export to RTF**: Download → Rich Text (.rtf)
- **Import RTF**: Upload → Select .rtf file
- Works with Microsoft Word, WordPad, LibreOffice

### PDF Import Support ✅
- **Import PDF**: Upload → Select .pdf file
- Extracts text from PDF documents
- Works with all standard PDF files

### DOC/DOCX Import Support ✅
- **Import DOC**: Upload → Select .doc or .docx file
- Basic text extraction from Word files
- For best results, save as RTF or TXT first

## ⌨️ Shortcuts

- `Ctrl+K` - Command palette
- `Ctrl+S` - Save document
- `Ctrl+/` - Focus mode
- `Esc` - Exit focus mode

## 🎯 Quick Tasks

### Transform Text
1. Type "hello world"
2. Click Transform → UPPERCASE
3. Result: "HELLO WORLD"

### Extract Emails
1. Paste text with emails
2. Click Extract → Extract Emails
3. View results in popup

### Export to RTF
1. Type your text
2. Download → Rich Text (.rtf)
3. Open in WordPad/Word

### Compare Texts
1. Click Compare tab
2. Paste two texts
3. Click Compare
4. View differences

## 🐛 Troubleshooting

### Upload Not Working?
- Check file format (.txt, .html, .rtf, .json, .md)
- Try a different file
- Refresh page (Ctrl+F5)

### Download Not Working?
- Check browser download settings
- Try different format
- Refresh page

### Stats Not Updating?
- Type some text
- Refresh page
- Check browser console (F12)
